Адаптер сигналов ВЗЛЕТ АС. Предназначен для построения беспроводных систем учета и диспетчеризации. Подключение к приборам осуществляется по интерфейсам RS-232 и RS-485, к диспетчерскому компьютеру — через среду сотовой связи и Интернет.

===

 _**Адаптирован для использования в составе АСКУТЭ ООО "Петербургтеплоэнерго".**_

![](assb-31.jpg)

<div class="property description" umi:element-id="1783" umi:field-name="description"><p>В качестве программного ядра диспетчерских систем используется программный комплекс <strong><em>ВЗЛЕТ СП</em></strong>. Диспетчерский компьютер должен быть доступен из Интернета по фиксированному IP-адресу. В отличие от циклического опроса, когда обмен в каждый момент времени осуществляется только с одним прибором, TCP/IP соединения, осуществляемые адаптерами, позволяют одновременно взаимодействовать с сотнями приборов. Фактическим ограничением является пропускная способность канала подключения к Интернету.</p>
<p>Диспетчерская система, построенная на основе адаптеров АССВ-031 ПР, является одним из вариантов информационно — измерительной системы ВЗЛЕТ ИИС, внесенной в Государственный реестр СИ, что делает возможным ее применение и для коммерческих расчетов.</p>
<h3>Функциональные возможности:</h3>
<ul class="dash">
<li>обеспечение сеансового удаленного доступа к узлам учета для контроля измерений в режиме реального времени с отображением данных в виде диаграмм, таблиц, мнемосхем, отчетов и т.п.;</li>
<li>передача накопленных данных для автоматической подготовки коммерческих отчетов и анализа работы узлов учета (сопоставление температурных графиков, выявление аномального потребления и т.п.);</li>
<li>оперативное информирование о нештатных ситуациях в измерениях и о состоянии узла учета в целом (охранная, пожарная сигнализация, затопление и т.п.).</li>
</ul>
<h3>Отличительные особенности:</h3>
<ul class="dash">
<li>низкие эксплуатационные расходы при использовании GPRS-услуги сотовых сетей (по сравнению с традиционным применением сотовых модемов на основе CSD услуги затраты снижаются в 50-100 раз);</li>
возможность передачи данных трем пользователям;
<li>визуальное отображение режимов работы;</li>
<li>оснащен четырьмя дискретными входами для подключения извещателей (охранных, пожарных и т.п.);</li>
<li>тип обслуживаемого прибора определяется драйвером, загруженным в энергонезависимую память адаптера;</li>
</ul>
<h3>Применение с приборами:</h3>
<ul class="dash">
<li><strong>тепловычислители:</strong><br>ВЗЛЕТ ТСРВ, СПТ-941, -941.11, -941.20, -942, -943, -943.1, -944, -961, -961М, -961.1, -961.2, -962, ВКТ-7, КМ-5, Эльф, ИМ2300, ЭСКО-Т, ЭСКО-ТМ (МТР-06), ТЭМ-104, ТВ7, ТС-11, Multical 602, ВКТ-5;</li>
<li><strong>расходомеры:</strong><br>ВЗЛЕТ ЭР, ВЗЛЕТ ЭМ, ВЗЛЕТ ТЭР, ВЗЛЕТ МР, ВЗЛЕТ РСЛ, ВЗЛЕТ РСЛ-2хх, ВЗЛЕТ РБП;</li>
<li><strong>газовые корректоры:</strong><br>ВЗЛЕТ КГ, СПГ-741, -761, -761.1, -761.2, -762, -763;</li>
<li><strong>электросчетчики, сумматоры:</strong><br>Меркурий-230, СЭТ-4ТМ.03, ПСЧ-ЗАРТ.07, СПЕ-542, ЭУ20М;</li>
<li><strong>другие приборы:</strong><br>ВЗЛЕТ УР-2хх, ВЗЛЕТ РО-2, РО-2М,&nbsp;РО-2 вент, ВЗЛЕТ РТ, КИВ ВЗЛЕТ (ИВК-101, -102, -102П,&nbsp;-103), АПС-79.</li>
</ul></div>
