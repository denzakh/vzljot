Доступное решение для использования на большинстве объектов для абонентского учета тепла с точностью, регламентируемой для коммерческого учета.

Обладает повышенной защитой от несанкционированного доступа.

Является лучшим в своем классе решением для общедомового коммерческого учета с возможностью включения в систему диспетчеризации.
