Специализированный электромагнитный расходомер-счетчик для систем поддержания пластового давления.

Предназначен для измерения в трубопроводах высокого давления среднего объемного расхода и объема жидких сред (минерализованной оборотной воды, пресной воды).

===

### Варианты исполнения:
* ППД-113, ППД-Ex – без индикатора;  
* ППД-213 – с индикатором.  

<h3>Относительная погрешность измерения:</h3>
<ul class="dash">
<li>от ±1,0 до ±2,0% - при температуре окружающей среды от 0 до 50 °С;</li>
<li>от ±2,0 до ±3,0% - при температуре окружающей среды от минус 40 до 0 °С.</li>
</ul>
<h3>Отличительные особенности:</h3>
<ul class="dash">
<li>полнопроходные расходомеры без потери давления на измерительном участке;</li>
<li>специальные конструкционные материалы, обеспечивающие высокую надежность в течение всего срока эксплуатации;</li>
<li>конструктивная взаимозаменяемость при монтаже с широко распространенными вихревыми расходомерами;</li>
<li>возможность использования во взрывоопасных зонах;</li>
<li>измерение расхода и объема реверсивного потока (по заказу).</li>
</ul>
<h3>Вывод информации:</h3>
<ul class="dash">
<li>на жидкокристаллический индикатор (для ППД-213);</li>
<li>по последовательному интерфейсу RS-485;</li>
<li>в виде импульсов с нормированным весом;</li>
<li>в виде нормированного токового сигнала (по заказу, кроме ППД-Ех).</li>
</ul>
